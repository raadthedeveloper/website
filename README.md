# Raad Fakhrian's Portfolio

A modern, responsive portfolio website built with Next.js and Tailwind CSS.

## Features

- Responsive design
- Modern UI with smooth transitions
- Blog section
- Project showcase
- Contact form
- SEO optimized

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- React
- Node.js

## Getting Started

### Prerequisites

- Node.js 18.0 or later
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <your-repo-url>
```

2. Navigate to the project directory:
```bash
cd website
```

3. Install dependencies:
```bash
npm install
# or
yarn install
```

### Development

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Building for Production

```bash
npm run build
# or
yarn build
```

Then start the production server:

```bash
npm run start
# or
yarn start
```

## Project Structure

```
website/

│   ├── components/       # Reusable components
│   └── styles/          # Global styles
├── public/              # Static assets
├── package.json

```

## Customization

1. Update the personal information in the components
2. Replace the project images in the public directory
3. Modify the color scheme in the Tailwind config
4. Add your own content to the blog section
